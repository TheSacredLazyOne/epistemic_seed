# integrated/

This directory contains **integrated references**: materials closer to embodied consequence.

In this repository, “integrated” means:
- nearer the consequence-bearing interface (the embodied prosthetic)
- evidence, constraints, and ground-truth surfaces
- things that *push back* on abstraction

Integrated references are included in frame bundles (e.g., `--bundle all`) to tether derivative reasoning to constraint.

Notes:
- This directory does not grant authority by itself.
- It exists to make constraint legible and accessible.
- If a referenced node is external, include a pointer file (URL + commit hash) or link it as a submodule/subtree under custody.
