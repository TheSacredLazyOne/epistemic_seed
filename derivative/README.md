# derivative/

This directory contains **derivative references**: materials extending away from the integration point.

In this repository, “derivative” means:
- abstractions, models, interpretations, and tooling
- proposals, drafts, experiments, and extensions
- structures that may later be revised, merged, or forked

Derivative references are included in frame bundles (e.g., `--bundle derivative` or `--bundle all`) as the “amplifier space” where reasoning and synthesis occur.

Notes:
- Derivative work remains coupled to constraint through tethered contact.
- Derivative does not imply “less true”; it implies “more revisable.”
- Divergence is expected. Forks are legitimate.
